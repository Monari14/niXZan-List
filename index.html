Descrição

niXZan - List é um aplicativo Android desenvolvido para gerenciar tarefas ou itens em uma lista. Com uma interface simples, o usuário pode adicionar, marcar ou remover itens da lista. O app utiliza SQLite para armazenar dados localmente, garantindo persistência, e funciona de maneira intuitiva, permitindo que o usuário interaja facilmente com a aplicação.

Funcionalidades

Adicionar Itens: O usuário pode adicionar itens à lista com uma descrição.

Marcar como Concluído: Itens podem ser marcados como concluídos, utilizando um checkbox.

Remover Itens: O usuário pode excluir itens da lista.

Persistência de Dados: Itens marcados como concluídos permanecem marcados mesmo após o app ser fechado e reaberto.

Armazenamento Local: O aplicativo utiliza SQLite para gerenciar os dados de maneira eficiente e local.

Estrutura do Banco de Dados

O banco de dados é estruturado em uma única tabela chamada items. Abaixo estão as colunas dessa tabela:

Tabela: items

Coluna Tipo de Dados Descrição
id INTEGER Identificador único do item (Chave Primária).
description TEXT Descrição do item (por exemplo, “Comprar café”).
checked INTEGER Indica se o item foi marcado como concluído (0 = não, 1 = sim).
createdAt TEXT Data de criação do item.

Funções do Banco de Dados

addItem(description, checked) - Adiciona um novo item à tabela items com a descrição fornecida e o estado de "checado" (concluído).

getAllItems() - Retorna todos os itens armazenados no banco de dados. Cada item é representado por um objeto ItemModel.

updateChecked(id, checked) - Atualiza o status de "checado" de um item existente na tabela.

deleteItem(id) - Deleta um item da tabela items.

Fluxo do Aplicativo

Adicionar Itens: O usuário pode inserir um novo item na lista ao preencher um campo de descrição e clicar em "Adicionar".

Marcar como Concluído: O usuário pode marcar um item como concluído ao selecionar a caixa de seleção ao lado do item.

Excluir Itens: O usuário pode excluir um item da lista clicando no ícone de lixeira ao lado do item.

Persistência: Quando o usuário fecha o aplicativo, as alterações feitas (como itens concluídos) são salvas no banco de dados e recuperadas quando o app é reaberto.

Futuras Expansões

Edição de Itens: A funcionalidade de editar os itens da lista será adicionada em uma futura versão.
